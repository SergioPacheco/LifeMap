# Índice — Coleção AstrologyPDF (Hermes Astrology)

> Coleção de 28 mini-guias da **Hermes Astrology** que cruzam livros populares de desenvolvimento pessoal com a linguagem astrológica.
> Todos em inglês. Avaliação feita em 2026-07-05.

---

## O que é esta coleção

Diferente dos PDFs técnicos italianos da LunaSapiens, esta coleção **não ensina astrologia técnica**. Em vez disso, usa a astrologia como linguagem interpretativa de temas psicológicos universais: propósito de vida, emoções, cura de traumas, liderança, liberdade, relacionamentos.

**Formato:** cada PDF tem ~6–8 páginas + página "About the Author" + oferta de newsletter gratuita.

**Uso para o LifeMap Pro:** rico em **linguagem interpretativa**, **conexões planeta-arquétipo** e **copy para relatórios**. Não serve como referência técnica de cálculo.
